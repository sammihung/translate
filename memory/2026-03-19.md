## Issues Fixed

### 1. Silero VAD Model Path Error
**Problem:** `FileNotFoundError: VAD 模型不存在：silero_vad.onnx`

**Root Cause:** The VAD model was in `.venv\Lib\site-packages\silero_vad\data\` but the code was looking in the current working directory.

**Fix:** Modified `vad_processor.py` to automatically load the model from the silero-vad package directory:
```python
import silero_vad
package_dir = Path(silero_vad.__file__).parent
self.model_path = str(package_dir / "data" / "silero_vad.onnx")
```

Also copied model files from old `.venv` to new `.venv`.

### 2. VAD ONNX Model Input Error
**Problem:** `Required inputs (['state', 'sr']) are missing from input feed (['input'])`

**Root Cause:** Silero VAD ONNX model requires 3 inputs: `input`, `state`, and `sr` (sample rate), but code only provided `input`.

**Fix:** Completely rewrote `vad_processor.py` to:
- Initialize RNN state as `np.zeros((2, 1, 128), dtype=np.float32)`
- Pass all 3 inputs: `input`, `state`, `sr`
- Update state after each inference
- Added `reset()` method to reinitialize state

### 3. Audio Device [Errno -9999] Error
**Problem:** Stereo Mix device failed with `[Errno -9999] Unanticipated host error`

**Root Cause:** Stereo Mix is disabled by default in Windows sound settings.

**Working Devices Found:**
- Device 7: Microphone Array (Intel Smart Sound) - works at 48000Hz
- Device 11/12: Stereo Mix - disabled in Windows

**Fix:** Added fallback audio stream initialization in `app.py` to:
- Try 16000Hz first
- Fall back to device default sample rate
- Fall back to 48000Hz stereo for Stereo Mix

**To Enable Stereo Mix:** Run `mmsys.cpl` → Recording tab → Enable Stereo Mix

### 4. ASR Engine None Error
**Problem:** `'NoneType' object has no attribute 'process_audio'`

**Root Cause:** ASR model directory `ov_models/qwen3_asr_int8` doesn't exist (never downloaded/copied).

**Fix:** Modified `app.py`:
- Made ASR engine optional in `init_engines()`
- Added None check in `process_audio_buffer()` before calling `asr_engine.process_audio()`
- Shows warning "⚠️ ASR 模型未載入，跳過語音辨識" when model is missing
