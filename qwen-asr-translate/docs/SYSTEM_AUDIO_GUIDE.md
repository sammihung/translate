# 🎵 系統音源設定指南

## 目標
讓 QwenASR 可以收聽電腦內部發出的聲音（YouTube、音樂、影片等），而不是麥克風。

---

## 方法 1: 啟用 Windows 立體聲混音 (最推薦)

### 適用於
Windows 10/11 大多數電腦

### 步驟

#### 1. 開啟音效設定
- 右鍵點擊工作列右下角的 **喇叭圖示**
- 選擇 **「聲音」** 或 **「音效內容」**

或者：
- 按下 `Win + R` 輸入 `mmsys.cpl` 按 Enter

#### 2. 找到錄音裝置
- 切換到 **「錄音」** 分頁
- 你會看到麥克風等錄音裝置列表

#### 3. 啟用立體聲混音
- 在空白處 **右鍵點擊**
- 勾選：
  - ✅ **顯示停用的裝置**
  - ✅ **顯示已中斷的裝置**

#### 4. 尋找「立體聲混音」
可能名稱：
- **立體聲混音** (中文)
- **Stereo Mix** (英文)
- **Wave Out**
- **What U Hear**

#### 5. 啟用並設為預設
- 右鍵點擊「立體聲混音」
- 選擇 **「啟用」**
- 再次右鍵 → **「設為預設裝置」**

#### 6. 測試
- 對著立體聲混音說話（播放音樂）
- 綠色的音量條應該會跳動

#### 7. 在 QwenASR 中使用
- 開啟程式
- 點擊「🔄 重新整理」
- 選擇「立體聲混音 🎵 (系統音源)」
- 開始錄音！

## 方法 2: 安裝虛擬音訊線 (萬一沒有立體聲混音)

### 為什麼需要？
部分新電腦或筆電移除了立體聲混音功能，需要用虛擬音訊線模擬。

### 推薦軟體

#### 選項 A: VB-Audio Virtual Cable (最簡單)
- **免費**
- **輕量** (只佔幾 MB)
- **簡單易用**

**下載**: https://vb-audio.com/Cable/

**安裝步驟**:
1. 下載 ZIP 檔案
2. 解壓縮
3. 右鍵 `VBCABLE_Setup_x64.exe` → **以系統管理員身分執行**
4. 安裝完成後 **重啟電腦**

**設定**:
1. 右鍵喇叭圖示 → 聲音
2. **「播放」** 分頁 → 設為預設：
   - **CABLE Input (VB-Audio Virtual Cable)**
3. **「錄音」** 分頁 → 確認有：
   - **CABLE Output (VB-Audio Virtual Cable)**
4. 回到 QwenASR → 選擇 **CABLE Output**

**使用方式**:
```
電腦播放 → Virtual Cable → QwenASR 收聽
```

#### 選項 B: VoiceMeeter (進階，可混音)
- **免費/付費**
- **功能強大**
- **可同時收聽麥克風 + 系統音**

**下載**: https://vb-audio.com/Voicemeeter/

**版本選擇**:
- **VoiceMeeter** (基本版，免費) - 夠用
- **VoiceMeeter Banana** (進階版，付費) - 專業需求

**基本設定**:
1. 安裝後重啟
2. 開啟 VoiceMeeter
3. **A1** 輸出 → 選擇你的耳機/喇叭
4. **Virtual Input** → 自動接收系統聲音
5. QwenASR 選擇 **VoiceMeeter Output**

**優點**: 
- 可同時混合麥克風和系統音
- 可調整音量平衡
- 低延遲

## 方法 3: 音效卡內建迴環

### 適用於
獨立音效卡 (Realtek, Creative 等)

### Realtek 音效卡
1. 開啟 **Realtek HD Audio Manager**
2. 找到 **「錄音」** 或 **「音效輸入」**
3. 尋找 **「What U Hear」** 或 **「Mix Stereo Out」**
4. 啟用並設為預設

### Creative 音效卡
1. 開啟 **Creative Control Panel**
2. 選擇 **「Recorder」**
3. 勾選 **「What U Hear」**
4. 設為預設錄音裝置

## 驗證設定是否成功

### 測試步驟
1. **開啟 QwenASR**
2. **選擇系統音源裝置** (立體聲混音或虛擬線)
3. **點擊「▶ 開始錄音」**
4. **播放電腦聲音**:
   - YouTube 影片
   - Spotify/音樂
   - 遊戲音效
5. **觀察**:
   - 音量條應該跳動
   - 即時字幕應該出現辨識結果

### 成功指標
✅ 音量條隨聲音跳動  
✅ 即時字幕正確顯示  
✅ 無爆音或斷續  

### 常見問題

#### ❌ 找不到立體聲混音
**解法**: 使用方法 2 安裝 Virtual Cable

#### ❌ 有畫面但無聲音
**解法**: 
- 確認系統音源已設為「預設裝置」
- 調高系統音量
- 重啟程式

#### ❌ 爆音或失真
**解法**:
- 降低系統音量到 70-80%
- 檢查取樣率是否為 44100Hz 或 48000Hz

#### ❌ 延遲過高
**解法**:
- 使用立體聲混音（延遲最低）
- 避免使用多層虛擬設備
- 關閉其他音訊軟體

## 進階應用

### 同時收聽麥克風 + 系統音

**需要**: VoiceMeeter 或類似混音器

**設定**:
1. 麥克風 → VoiceMeeter Input 1
2. 系統音 → VoiceMeeter Virtual Input
3. QwenASR → VoiceMeeter Output
4. 調整兩者音量平衡

### 錄製特定應用程式聲音

**Windows 10/11 內建**:
1. 設定 → 系統 → 音效
2. 應用程式音量和裝置偏好設定
3. 個別調整應用程式輸出

### 多國語言影片即時翻譯

**情境**: YouTube 日文影片 → 中文字幕

**步驟**:
1. 設定系統音源
2. QwenASR 選擇：
   - 辨識語言：**日文**
   - 翻譯語言：**繁體中文**
3. 播放影片
4. 即時看到雙語字幕！

## 效能優化

### 降低延遲
1. 使用 **立體聲混音** (延遲 <100ms)
2. 取樣率設為 **48000Hz**
3. 緩衝區設為 **512** 或 **1024** samples

### 提升辨識準確度
1. 確保音源清晰（避免背景噪音）
2. 音量適中（-12dB 到 -6dB）
3. 使用 1.7B 模型（如有 GPU）
