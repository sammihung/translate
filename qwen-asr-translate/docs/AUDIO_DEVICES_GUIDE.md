```
選擇：🎵 立體聲混音 或 🎵 CABLE Output

```

```
選擇：🎵 立體聲混音

```
選擇：麥克風陣列

## 常見問題

### Q: 如何確認我選對了裝置？

**測試方法**:
1. 選擇裝置後點擊「▶ 開始錄音」
2. 播放電腦聲音（音樂/YouTube）
3. 觀察音量條是否跳動
4. 看即時字幕是否有內容

✅ **有跳動 + 有字幕** = 正確  
❌ **無跳動** = 選錯裝置或未啟用  

### Q: 為什麼沒有「立體聲混音」選項？

**原因**: 
- 部分新電腦移除此功能
- 驅動程式未安裝完整
- 需要手動啟用

**解決方案**:
1. 執行 `setup-audio.bat`
2. 選擇選項 2 安裝 Virtual Cable
3. 重啟電腦後再試

### Q: 同時收聽麥克風和系統音？

**需要**: VoiceMeeter 混音器

**設定**:
1. 安裝 VoiceMeeter
2. 麥克風 → VoiceMeeter Input 1
3. 系統音 → VoiceMeeter Virtual Input
4. QwenASR → VoiceMeeter Output
5. 調整音量平衡

## 快速切換

### 在 QwenASR 中切換裝置

1. 點擊「🔄」重新整理裝置列表
2. 在下拉選單選擇新裝置
3. 無需重啟程式，立即生效！

### 建議

- **收聽電腦**: 立體聲混音/Virtual Cable
- **收聽說話**: 麥克風陣列/USB 麥克風
- **混合使用**: VoiceMeeter
