## 互動效果

### 按鈕狀態
- **常態**: 實心顏色，白色文字
- **懸停**: 亮度 +10%
- **按下**: 亮度 -10%
- **停用**: 灰色，50% 透明度

### 進度條
- 平滑動畫過渡
- 百分比數字即時更新
- 完成時綠色對勾動畫

### 狀態圖示
- 🟢 就緒
- 🔴 錄音中/錯誤
- 🟡 處理中/警告
- 🔵 等待中/資訊
