⚠️ 第一次使用？請依照以下步驟：

這個安裝程式會自動下載並安裝 Python 3.12，然後安裝 QwenASR 應用程式。整個過程約需 10-20 分鐘。

1全自動安裝（推薦）

點擊下方按鈕會自動下載並安裝 Python 3.12，然後安裝 QwenASR：

🚀 點擊這裡開始全自動安裝

💡 這個程式會自動完成所有步驟，包括下載 Python、建立虛擬環境、安裝依賴套件

2手動安裝（如果自動安裝失敗）

如果自動安裝失敗，請手動執行以下步驟：

2.1 下載 Python 3.12

📥 前往 Python.org 下載

2.2 安裝 Python

  • 執行下載的安裝檔
  • ⚠️ 重要：勾選 "Add Python to PATH"
  • 點擊 "Install Now"
  • 等待安裝完成
  • 重啟電腦

2.3 安裝 QwenASR

Python 安裝完成後，雙擊執行：

INSTALL.bat

或執行：

install-and-run.bat

3開始使用

安裝完成後會自動啟動應用程式。之後使用只需雙擊：

quick-start.bat

✅ 安裝完成後您將獲得：

• 即時語音辨識（麥克風或系統音源）
• 即時翻譯（30+ 語言）
• 雙語字幕顯示
• 支援 YouTube、音樂、影片轉錄
• 支援 MP3/WAV/FLAC/MP4/MKV 檔案
