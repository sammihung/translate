## 🚨 第一次使用？請依照以下步驟：

### 步驟 1: 安裝 Python（必須！）

如果您的電腦**還沒安裝 Python**，請執行：

1. 開啟瀏覽器
2. 前往：**https://www.python.org/downloads/**
3. 下載 **Python 3.12** (最新版本)
4. 執行安裝檔
5. **⚠️ 重要**: 勾選 ✅ **"Add Python to PATH"**
6. 點擊 **"Install Now"**
7. 等待安裝完成
8. **重啟電腦**

### 步驟 2: 一鍵安裝

Python 安裝完成後，**雙擊執行**:

```
INSTALL.bat
```

這個腳本會自動：
- ✅ 檢查 Python 是否已安裝
- ✅ 檢查 Python 版本（需要 3.10+）
- ✅ 建立 `.venv` 虛擬環境
- ✅ 安裝所有依賴套件
- ✅ 自動啟動應用程式

**安裝時間**: 約 5-15 分鐘（視網路速度）

### 步驟 3: 開始使用

安裝完成後會自動啟動應用程式。

之後使用，只需**雙擊**:

```
quick-start.bat
```

## 🎯 主要功能

✅ **即時語音辨識** - 麥克風或系統音源  
✅ **即時翻譯** - 30+ 語言  
✅ **雙語字幕** - 原文 + 譯文並排  
✅ **系統音源** - 收聽 YouTube、音樂、影片  
✅ **檔案轉換** - MP3/WAV/FLAC/MP4/MKV  
✅ **OpenVINO INT8** - CPU 高效推理  
✅ **GPU 加速** - Vulkan 支援（可選）  
✅ **一鍵安裝** - 自動化 `.venv` 管理

## 🛠️ 系統需求

### 必要
- Windows 10/11 (64-bit)
- **Python 3.10+** (需要手動安裝)
- 6 GB RAM
- 2 GB 硬碟空間

- 8 GB RAM
- GPU (NVIDIA/AMD/Intel)

## 🔧 常見問題

### Q: 雙擊腳本沒反應？
**A**: Python 未安裝或未加入 PATH  
**解法**: 執行 `INSTALL.bat` 會自動檢查並引導

### Q: 找不到 Python？
**A**: 
1. 重新安裝 Python
2. 勾選 "Add Python to PATH"
3. 重啟電腦

### Q: 如何設定收聽電腦聲音？
**A**: 執行 `setup-audio.bat` 並跟隨指示
