# QwenASR Translate - 打包指南

## 📦 方法對比

| 方法 | 大小 | 專業度 | 難度 | 推薦 |
|------|------|--------|------|------|
| **ZIP 資料夾** | ~2.5GB | ⭐⭐ | 簡單 | ✅ 快速測試 |

## 🚀 方法 1: ZIP 資料夾 (快速)

### 步驟：

1. **執行打包**：
   ```bash
   .\build_exe.bat
   ```

2. **壓縮資料夾**：
   - `dist\QwenASR Translate\` → 右鍵 → 傳送到 → 壓縮資料夾

3. **分享**：
   - Upload 去 Google Drive / OneDrive
   - 附上 `給用戶的說明.txt`

### 優點：
- ✅ 簡單快速
- ✅ 無須額外軟件

### 缺點：
- ❌ 無安裝程式
- ❌ 無桌面捷徑
- ❌ 無解除安裝
- ❌ 體積較大

## 🏆 方法 2: Inno Setup 安裝檔 (專業)

### 前置作業：

**1. 下載 Inno Setup**：
```
https://jrsoftware.org/isdl.php
下載：innosetup-6.x.exe
安裝：一路 Next 就得
```

**2. 修改 pyproject.toml** (已完成)：
- 移除 `pyannote-audio` (減少 ~500MB)
- 移到 `[project.optional-dependencies]`

### 步驟：

**1. 執行完整打包**：
```bash
.\build_full.bat
```

**2. 等待完成** (約 15-20 分鐘)：
- PyInstaller 打包 (~10 分鐘)
- Inno Setup 打包 (~5 分鐘)

**3. 檢查輸出**：
```
installer_output\QwenASR-Translate-Setup-0.1.0.exe
```

**4. 測試安裝檔**：
- 雙擊 `.exe` 安裝檔
- 跟足安裝步驟
- 測試程式是否正常運行

### 安裝檔功能：

✅ **自動創建**：
- 桌面捷徑
- 開始功能表
- 解除安裝程式

✅ **安裝位置**：
- 預設：`C:\Program Files\QwenASR Translate\`
- 用戶可自訂

✅ **系統整合**：
- 新增/移除程式集
- 檔案關聯 (可選)

✅ **多語言支援**：
- 英文
- 繁體中文
- 簡體中文

### 優點：
- ✅ 專業安裝體驗
- ✅ 自動創建捷徑
- ✅ 完整解除安裝
- ✅ 體積較小 (壓縮更好)
- ✅ 系統整合

### 缺點：
- ❌ 需要 Inno Setup
- ❌ 打包時間較長

## 🔧 進一步優化

### 1. 移除更多依賴 (可選)

編輯 `pyproject.toml`：

```toml
# 移除 gradio (如果無用到 GUI 測試)
# "gradio>=5.0.0"

# 移除 lightning (如果無用到訓練)
# "lightning>=2.0.0"
```

### 2. 使用 UPX 壓縮

修改 `qwen_asr.spec`：

```python
exe = EXE(
    ...
    upx=True,  # 啟用 UPX 壓縮
    upx_exclude=[],  # 排除特定檔案
)
```

可再減少 **30-40%** 體積！

### 3. 分割安裝檔

如果需要網絡分發，可以分割成多個部分：

```bash
# 使用 7-Zip 分割
7z a -v500m QwenASR.7z dist\QwenASR Translate\
```

生成：
- `QwenASR.7z.001` (500MB)
- `QwenASR.7z.002` (500MB)
- `QwenASR.7z.003` (300MB)

## ⚠️ 注意事項

### 1. 首次運行仍然需要下載模型

用戶首次執行時：
- ASR 模型：~3-5GB (HuggingFace)
- 翻譯模型：~2-4GB (Ollama)

**建議**：
- 在說明中清楚寫明
- 提供離線模型下載連結 (可選)

### 2. Ollama 必須預先安裝

安裝檔會提示用戶，但唔會自動安裝 Ollama。

**原因**：
- Ollama 係獨立服務
- 需要用戶同意安裝
- 模型文件好大 (唔適合打包)

### 3. 防毒軟件誤判

PyInstaller 打包嘅 .exe 可能觸發防毒警告。

**解決方案**：
- 提供數位簽章 (需購買證書)
- 在說明中提醒用戶
- 提供 SHA256 哈希值驗證

## 📞 疑難排解

### Q: Inno Setup 找不到？
**A**: 確保安裝在預設路徑：
```
C:\Program Files (x86)\Inno Setup 6\ISCC.exe
```

### Q: 打包失敗？
**A**: 檢查：
1. `dist\QwenASR Translate\` 是否存在
2. `installer.iss` 路徑是否正確
3. 磁碟空間是否足夠 (至少 5GB)

### Q: 安裝後無法運行？
**A**: 檢查：
1. Ollama 是否安裝
2. 防毒軟件是否封鎖
3. 以管理員身份運行
