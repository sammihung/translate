# 🎵 系統音源設定總覽

## 快速設定（3 分鐘完成）

```
┌─────────────────────────────────────────────────────────┐
│  🎯 目標：讓 QwenASR 收聽電腦內部的聲音                  │
└─────────────────────────────────────────────────────────┘

方法 1: 啟用立體聲混音 (如果有)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
右鍵喇叭 → 聲音 → 錄音 → 啟用立體聲混音 → 完成！

方法 2: 安裝 Virtual Cable (萬一沒有)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
下載 → 安裝 → 重啟 → 選擇 CABLE Output → 完成！

方法 3: 使用設定精靈 (最簡單)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
雙擊 setup-audio.bat → 跟隨指示 → 完成！
```

## 裝置選擇決策樹

```
你想收聽什麼聲音？
│
├─ 電腦內部的聲音
│  (YouTube、音樂、影片、遊戲)
│  │
│  ├─ 有「立體聲混音」？
│  │  ├─ YES → 選擇「立體聲混音 🎵」✅
│  │  └─ NO  → 安裝 Virtual Cable
│  │           → 選擇「CABLE Output 🎵」✅
│  │
│  └─ 進階需求（混音）？
│     └─ 是 → 安裝 VoiceMeeter
│             → 選擇「VoiceMeeter Output 🎵」✅
│
└─ 外部的聲音
   (說話、會議、訪談)
   │
   ├─ 內建麥克風？
   │  └─ 選擇「麥克風陣列 🎤」✅
   │
   └─ 外接麥克風？
      └─ 選擇對應的 USB 麥克風 🎤✅
```

## 快速比較

| 功能 | 麥克風模式 | 系統音源模式 |
|------|-----------|-------------|
| **用途** | 收聽說話 | 收聽電腦聲音 |
| **裝置** | 麥克風陣列 | 立體聲混音 |
| **延遲** | <50ms | <100ms |
| **準確度** | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ |
| **適用場景** | 會議、訪談 | YouTube、音樂 |
