```

│  即時轉換分頁 │  上傳翻譯分頁 │  批次處理分頁 │   設定分頁     │

                              │

│   ASR 引擎        │   翻譯引擎        │   VAD 處理器         │
```

## 核心模組

### 1. ASR 引擎 (`asr_engine.py`)

**功能**: 語音辨識核心
- 使用 Qwen3-ASR 模型（OpenVINO INT8 量化）
- 支援 CPU (OpenVINO) 和 GPU (Vulkan) 後端
- Mel 頻譜計算與特徵提取
- BPE 解碼與文字輸出

**模型選項**:
- **0.6B INT8**: 1.2 GB，適合一般用途
- **1.7B INT8**: 4.3 GB，更高準確度

**推理流程**:
```
音訊輸入 → 重取樣 (16kHz) → Mel 頻譜 → OpenVINO 推理 → BPE 解碼 → 文字輸出
```

### 2. VAD 處理器 (`vad_processor.py`)

**功能**: 語音活動偵測
- 使用 Silero VAD ONNX 模型
- 自動偵測語音段落
- 靜音分段與剪輯
- 即時緩衝區處理

**參數**:
- `threshold`: 語音閾值（預設 0.5）
- `min_speech_duration`: 最小語音長度（0.25 秒）
- `min_silence_duration`: 最小靜音長度（0.5 秒）

**工作流程**:
```
連續音訊 → 分幀 (40ms) → VAD 機率 → 閾值判斷 → 語音段落邊界
```

### 3. 翻譯引擎 (`translate_engine.py`)

**功能**: 即時文字翻譯
- 使用 OPUS-MT 神經機器翻譯模型
- 支援 30+ 語言對
- 輕量化設計（~100-300 MB 每語言對）
- 批次翻譯優化

**模型**: Helsinki-NLP/opus-mt-{src}-{tgt}

### 4. 字幕生成器 (`subtitle_generator.py`)

**功能**: 雙語字幕生成
- SRT/VTT 格式輸出
- 雙語並排顯示
- 時間軸自動對齊
- 說話者標記支援

**輸出格式**:
```srt
1
00:00:01,000 --> 00:00:05,000
原文：你好，歡迎來到今天的節目
譯文：Hello, welcome to today's show

說話者 1: 你好
說話者 2: 你好
```

### 5. GUI 介面 (`app.py`)

**功能**: 使用者介面
- CustomTkinter 現代化介面
- 深色/淺色模式切換
- 即時字幕顯示
- 雙語字幕預覽

**分頁結構**:
1. **即時轉換**: 麥克風輸入，即時辨識翻譯
2. **音檔轉字幕**: 檔案上傳，批次處理
3. **設定**: 外觀、模型、語言設定

## 音訊處理流程

### 即時模式

```
┌─────────────┐
│ 麥克風輸入   │
│ (16kHz)     │
└──────┬──────┘
       │
┌──────▼──────┐
│ 音訊緩衝區   │ (2 秒滾動緩衝)
└──────┬──────┘
       │
┌──────▼──────┐
│ VAD 偵測      │ (Silero)
└──────┬──────┘
       │
    [語音段落？]
       │
    是 ├──────→┌─────────────┐
       │       │ ASR 推理     │ (OpenVINO)
       │       └──────┬──────┘
       │              │
       │       ┌──────▼──────┐
       │       │ 翻譯引擎     │ (OPUS-MT)
       │       └──────┬──────┘
       │              │
       │       ┌──────▼──────┐
       │       │ 字幕顯示     │ (GUI)
       │       └─────────────┘
       │
    否 └──────→ 繼續監聽
```

### 檔案模式

```
┌─────────────┐
│ 音訊/影片檔 │
└──────┬──────┘
       │
┌──────▼──────┐
│ ffmpeg 提取  │ (影片 → 音訊)
└──────┬──────┘
       │
┌──────▼──────┐
│ 全段 VAD     │ (分段)
└──────┬──────┘
       │
┌──────▼──────┐
│ 批次 ASR     │ (平行處理)
└──────┬──────┘
       │
┌──────▼──────┐
│ 批次翻譯     │
└──────┬──────┘
       │
┌──────▼──────┐
│ SRT 匯出     │
└─────────────┘
```

## GPU 加速

### Vulkan 後端 (`app_gpu.py`)

使用 chatllm.cpp Vulkan 推理引擎：

**優勢**:
- 支援 NVIDIA/AMD/Intel GPU
- 無需 CUDA/ROCm
- 1.7B GGUF 模型 (~2.3 GB)
- 顯著提升準確度

**需求**:
- Vulkan 1.2+
- 8 GB+ RAM
- 2.3 GB 可用空間

### OpenVINO GPU

Intel GPU 可選 OpenVINO GPU 後端：
- 整合 Intel 顯示卡
- OneAPI 驅動
- 效能介於 CPU 與 Vulkan 之間

## 模型管理

### 目錄結構

```
qwen-asr-translate/
├── ov_models/
│   ├── qwen3_asr_int8/        # 0.6B INT8
│   │   ├── openvino_model.xml
│   │   └── openvino_model.bin
│   ├── qwen3_asr_1p7b_int8/   # 1.7B INT8 (選配)
│   ├── silero_vad.onnx        # VAD 模型
│   └── diarization/           # 說話者分離 (選配)
├── GPUModel/
│   └── qwen3-asr-1.7b.bin     # Vulkan GPU 模型
└── ffmpeg/
    └── ffmpeg.exe              # 影片處理 (自動下載)
```

### 下載機制 (`downloader.py`)

1. **互動式選單**: 選擇要下載的模型
2. **完整性檢查**: 啟動時自動驗證
3. **斷點續傳**: 支援中斷後繼續
4. **多來源**: HuggingFace/Gitee 鏡像

## 擴展性

### 新增語言
1. 下載對應 OPUS-MT 模型
2. 更新 `translate_engine.py` 語言列表
3. GUI 自動顯示新選項

### 新增模型
1. 轉換為 OpenVINO IR 格式
2. 添加到 `MODELS` 字典
3. 更新 `downloader.py`

### 自訂 UI
1. 繼承 `ctk.CTkFrame`
2. 實作自訂分頁
3. 註冊到 `MainWindow.tabview`

## 安全與隱私

### 本機處理
- 所有音訊處理均在本地
- 無需網路連線（除模型下載）
- 無資料上傳

### 模型驗證
- SHA256 雜湊檢查
- 檔案完整性驗證
- 來源可信認證

## 錯誤處理

### 異常類型
- 模型載入失敗
- 音訊裝置錯誤
- 記憶體不足
- 檔案格式不支援

### 復原機制
- 自動降級（GPU → CPU）
- 模型重新下載
- 裝置重新偵測
