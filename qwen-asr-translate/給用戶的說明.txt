========================================
  QwenASR Translate - 用戶使用指南
========================================

歡迎使用 QwenASR Translate！
即時語音辨識 + 翻譯工具

========================================
  第一步：解壓縮
========================================

1. 將下載嘅 ZIP 檔案解壓縮到任意位置
   建議：C:\Programs\QwenASR\

2. 確保解壓縮後有呢個檔案：
   - QwenASR Translate.exe
   - 其他資料夾（唔好改動）

========================================
  第二步：安裝必要軟件
========================================

【必須安裝】Ollama（用於翻譯）

1. 下載 Ollama：
   https://ollama.ai/download

2. 安裝後打開 Ollama（會喺背景運行）

3. 下載翻譯模型（可選，程式會自動下載）：
   打開命令提示字元，輸入：
   
   ollama pull translategemma:4b-it-q4_K_M

【可選安裝】VB-Audio Virtual Cable
（如果你要錄製 YouTube/系統聲音）

1. 下載：
   https://vb-audio.com/Cable/

2. 安裝後：
   - Windows 聲音設定 → 輸出設備選 "CABLE Input"
   - 應用程式 → 輸入設備選 "CABLE Output"

========================================
  第三步：執行程式
========================================

1. 雙擊 "QwenASR Translate.exe"

2. 首次運行可能需要 1-2 分鐘（下載模型）

3. 選擇效能模式：
   - ⚡ 極速版：CPU / 低階 GPU（推薦）
   - ⚖️ 平衡版：中階 GPU（GTX 1060+）
   - 🩸 滿血版：高階 GPU（RTX 3080+）

4. 選擇音訊輸入設備：
   - 麥克風：選你的麥克風
   - YouTube：選 "CABLE Output"

5. 開始錄音！

========================================
  常見問題
========================================

Q1: 程式打不開？
A1: 確保 Windows Defender 唔好封鎖 .exe
    右鍵 → 內容 → 解除封鎖

Q2: 翻譯唔工作？
A2: 確保 Ollama 已經安裝並運行
    打開命令提示字元輸入：ollama list
    應該見到 translategemma:4b-it-q4_K_M

Q3: 好慢？
A3: 
   - 如果用 CPU：正常，建議用極速模式
   - 如果用 GPU：確保 CUDA 已安裝
   - 關閉其他應用程式釋放 RAM

Q4: 找不到音訊設備？
A4: 
   - 重啟程式
   - 檢查 Windows 隱私權設定（麥克風權限）
   - 重新插拔麥克風

Q5: 首次運行好耐？
A5: 正常！程式會下載模型（約 2-5GB）
    之後運行會快好多

========================================
  系統需求
========================================

最低需求：
- Windows 10/11 (64 位元)
- 8GB RAM
- 10GB 儲存空間

建議需求：
- Windows 11
- 16GB RAM
- NVIDIA GPU (GTX 1060+)
- 20GB 儲存空間

========================================
  浮動字幕模式
========================================

1. 點擊 sidebar 嘅 "🪟 浮動字幕模式"

2. 浮動窗口會：
   - 永遠顯示在最前面
   - 半透明（唔阻視線）
   - 可以調較大小

3. 適合：
   - 睇 YouTube 影片
   - 網上會議
   - 直播字幕

========================================
  文字大小調整
========================================

點擊 sidebar 嘅 "🔤 文字大小" 按鈕：
- 小 → 標準 → 大 → 超大

所有字幕（包括浮動窗口）會即時更新！
